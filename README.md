# Homebrew Tap for FeedMe

This is the official Homebrew tap for [FeedMe](https://github.com/XuanLee-HEALER/FeedMe), a lightweight macOS menu bar RSS reader.

## Installation

```bash
brew tap XuanLee-HEALER/feedme
brew install --cask feedme
```

## Update

```bash
brew upgrade --cask feedme
```

## Uninstall

```bash
brew uninstall --cask feedme
brew untap XuanLee-HEALER/feedme
```

## About FeedMe

FeedMe is a native macOS menu bar RSS reader built with SwiftUI and AppKit.

- **Menu Bar Native** - Lives in your menu bar, minimal distraction
- **Left Click** - Quick access to latest articles
- **Right Click** - App settings and feed management
- **OPML Support** - Import/Export your subscriptions

For more information, visit the [main repository](https://github.com/XuanLee-HEALER/FeedMe).
